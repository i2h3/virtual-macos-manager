# Swift Package Template

This is my personal template repository for Swift packages.
It also serves as a reference for all the packages I maintain.

## Features

- Placeholder Swift source code files
- GitHub action for linting with SwiftFormat preconfigured
- GitHub action for automatic test target runs on various platforms preconfigured
- SwiftFormat preconfigured
- Basic AGENTS.md preconfigured
- MIT license prepared
